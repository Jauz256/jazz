# Internal Communications

## When to Use
- 3P updates (Progress, Plans, Problems)
- Company newsletters
- FAQ responses
- Status reports
- Leadership updates
- Project updates
- Incident reports

## How to Use

1. **Identify the communication type** from the request
2. **Load the appropriate guideline file** from the `examples/` directory
3. **Follow the specific instructions** for formatting, tone, and content

## Communication Types

- **3P Updates**: Progress/Plans/Problems team updates
- **Company Newsletter**: Company-wide newsletters
- **FAQ Answers**: Answering frequently asked questions
- **General Comms**: Anything else
