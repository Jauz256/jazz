# MengSpark Image Prompt Generator

Generate AI image prompts that produce consistent, on-brand visuals for MengSpark.

## Quick Start Formula

[STYLE] + [SUBJECT] + [COMPOSITION] + [BRAND ELEMENTS] + [TECHNICAL SPECS]

**Example:**
> Flat illustration style, friendly firefly character with purple body and glowing yellow abdomen, centered composition, MengSpark brand colors purple #8A2BE2 and gold #FCD34D, simple gradient background, clean vector art, 1080x1080

## Style Selection

| Use Case | Style | Key Prompt Words |
|----------|-------|------------------|
| Mascot illustrations | Flat | flat illustration, vector art, solid colors |
| Hero/marketing images | 3D Pixar | 3D rendered, Pixar style, soft lighting |
| App UI mockups | Glassmorphism | glassmorphism, frosted glass effect |
| Pitch deck scenarios | Semi-realistic | professional render, realistic lighting |

## Output Specifications

| Platform | Dimensions |
|----------|------------|
| Instagram Post | 1080×1080 |
| Instagram Story | 1080×1920 |
| Presentation Slide | 1920×1080 |
| App Store Screenshot | 1290×2796 |

## Prompt Checklist

- [ ] Style specified (flat/3D/glass/realistic)
- [ ] MengSpark colors included (hex codes)
- [ ] Composition/position keywords present
- [ ] Aspect ratio/dimensions stated
- [ ] Negative prompts added if needed
