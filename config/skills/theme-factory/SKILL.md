# Theme Factory

Professional font and color themes for presentations, docs, and artifacts.

## Usage

1. Show the theme showcase (theme-showcase.pdf)
2. Ask which theme to apply
3. Wait for selection
4. Apply the theme

## 10 Available Themes

1. **Ocean Depths** - Professional maritime theme
2. **Sunset Boulevard** - Warm sunset colors
3. **Forest Canopy** - Natural earth tones
4. **Modern Minimalist** - Clean grayscale
5. **Golden Hour** - Rich autumnal palette
6. **Arctic Frost** - Cool winter-inspired
7. **Desert Rose** - Soft dusty tones
8. **Tech Innovation** - Bold tech aesthetic
9. **Botanical Garden** - Fresh garden colors
10. **Midnight Galaxy** - Dramatic cosmic tones

## Theme Contents

Each theme includes:
- Cohesive color palette with hex codes
- Complementary font pairings (headers + body)
- Distinct visual identity

## Application Process

1. Read theme file from `themes/` directory
2. Apply colors and fonts consistently
3. Ensure proper contrast and readability
4. Maintain visual identity across all slides

## Custom Themes

If none work, create a custom theme:
1. Generate new theme based on description
2. Show for review
3. Apply after confirmation
